# 免费数据源结论

## 当前方案

采用双源策略：
- 主源：东方财富免费接口
- 备用：腾讯财经免费历史日 K

## 选择理由

### 东方财富
优点：
- 字段完整
- 前复权参数明确
- 适合直接生成回测输入

风险：
- 会有偶发性远端断连或反爬抖动

### 腾讯财经
优点：
- 免费
- 适合做备用拉取

风险：
- 接口格式和字段风格与东方财富不同，需要单独适配

## 当前环境验证

- 东方财富：曾成功返回真实数据，但存在偶发断连
- Tushare：当前环境未安装，不作为第一实现
- pandas：当前环境未安装，因此脚本优先使用 Python 标准库

## 实现原则

第一版只依赖 Python 标准库。

抓数失败时，直接报错，不补假数据。若主源失败，切备用源。两者都失败，再终止。
